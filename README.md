GramYatra is a village-first travel platform that connects travelers directly with rural India. 
 
Every booking supports: 
- Local homestays 
- Village guides 
- Regional food providers 
- Community development 
 
## Tech Stack 
- React + Vite 
- Node.js + Express 
- MongoDB 
- Firebase Storage 
- Tailwind CSS 
 
## Features 
- Village-based travel packages 
- Swadeshi impact tracking 
- User bookings & history 
- Admin impact dashboard 
- Local earnings distribution 
 
Built as part of an Atmanirbhar Bharat travel initiative.
